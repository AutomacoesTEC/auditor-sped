Capacidades

Tecnologia e Direito Tributario Integrados

Engenharia de dados aplicada a auditoria fiscal. Precisao normativa em cada linha de codigo.

Parser SPED Robusto

Le arquivos ECD e ECF com encoding ISO-8859-1/UTF-8. Parseia I050, I051, I155, I200, I250, N620, N630, N670, M300, Y570 e mais. 684K registros em ~10s.

Rastreabilidade Total

Cada achado aponta os registros SPED de origem: tipo de registro, linha do arquivo e valores. Auditavel pela Receita Federal.

22 Gatilhos Automaticos

12 de conta (GC01-GC12) e 10 de historico (GH01-GH10). Detectam padroes suspeitos em nomes de conta e descricoes de lancamentos.

Seguranca e Integridade

Protecao contra path traversal, validacao de entrada, sessoes isoladas por UUID, encoding com fallback gracioso. Zero crash em arquivo malformado.

Relatorios Profissionais

Gera Excel com 5 abas (por regra, por tributo, gatilhos) e Parecer Tecnico PDF com 7 secoes. Para entrega direta ao cliente.

Interface Web Flask

Upload via browser, execucao em um clique, download dos relatorios. Roda localmente na rede interna do escritorio.

Estrutura

Arquitetura Modular e Extensivel

Separacao estrita entre parsing, regras de auditoria, gatilhos e relatorios. Cada modulo e independente e testavel.

auditor-sped/
main.py # Orquestrador: parsing → regras → gatilhos → relatorios
config.py # Constantes, mapeamento de campos, linhas-chave ECF
requirements.txt # openpyxl, reportlab, flask

src/
parsers/
sped_parser.py # Parser generico: encoding, split por |
ecd_parser.py # ECD: I050, I051, I155, I200, I250
ecf_parser.py # ECF: N620, N630, N670, M300, Y570, Y580

regras/
base_regra.py # Interface abstrata RegraAuditoria + dataclass Achado
r01_saldo_negativo.py # Saldo neg. IRPJ/CSLL + IRRF nao utilizado
# r02 ... r10 # 9 regras adicionais implementadas

gatilhos/
motor_gatilhos.py # 12 gatilhos de conta (GC) + 10 de historico (GH)

relatorios/
gerador_xlsx.py # Excel: Capa, Resumo, por regra, gatilhos
gerador_pdf.py # Parecer tecnico PDF com 7 secoes

web/
app.py # Flask: upload, execucao, download
templates/ # base.html, index.html, resultado.html
Instalacao

Rodando em 3 passos

Python 3.11+ e dois pacotes. Sem banco de dados, sem Docker, sem infraestrutura complexa.

1
Clonar e instalar
Clone o repositorio e instale as dependencias.
git clone https://github.com/
AutomacoesTEC/auditor-sped
cd auditor-sped
pip install -r requirements.txt
2
Interface Web
Inicie o servidor e acesse pelo browser.
python web/app.py

# Acesse:
# http://localhost:5000
3
Linha de Comando
Ou execute direto pelo terminal com flags.
python main.py \
--ecd ecd.txt \
--ecf ecf.txt \
--xlsx --pdf \
--saida resultado/
